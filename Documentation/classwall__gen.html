◆ add_wall_chain()

void wall_gen::add_wall_chain ( const std::vector< sf::Vector2u > &  points)

used to add wall in chain fashion

enter a vector of sf::Vector2u points in player units and this function will create vertical/horizontal walls with width equal player length between consecutive
points in vector

Parameters
pointsvector of sf::vector2u points
Note
when constructing walls going downward/rightward direction ,their length extended by one to avoid cases of incomplete corner walls when the next wall segment is made
Warning
consecutive points must be on same vertical/ horizontal line ,or undefined behaviour
if 2 consecutive points are the same then a player sized block will be generated at that position
no output/change if only zero or one point in vector
