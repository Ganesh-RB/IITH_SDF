Detailed Description

class dealing with player movement,collision and markers

class for the player

this class handles player movement ,collision and placement of collectable markers which serve as victory condition

this class allows for the player's creation,movement, input and collisions while also creating the trails which colour the screen and the marker system which keeps track of whether a level is completely coloured or not, along with methods to add foresaid markers

void player::add_marker_chain ( const std::vector< sf::Vector2f > &  points)

used to add markers in chain fashion

enter a vector of sf::Vector2f points in player units and this function will create markers with spacing in the x and/or y direction equal to player length between consecutive points in vector

Parameters
pointsstd::vector<sf::Vector2f> vector of sf::vector2f points
Note
place markers in middle of grid squares
Warning
no output/change if only zero or one point in vector
if 2 consecutive points are the same then a single marker will be generated at that position
